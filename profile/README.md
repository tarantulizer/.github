# Tarantulizer - Scalable Web Scraping Framework

## Tools and Technologies
Here are the tools and technologies that makes up the **Tarantulizer Framework**.

### Back-End Technologies
* FastAPI
* Express.js

### Front-End Technologies
* Vue.js

### Message Broker
* Kafka

### Web Scraping Tools
* Selenium
* Puppeteer
* Playwright
* Scrappy
* Beautiful Soup

### DevOps Tools
* Docker
* Kubernetes
* Terraform

### CI / CD Tools
* Github Actions

### Cloud Platform
* Amazon AWS
